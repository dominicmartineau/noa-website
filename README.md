# noa - AI Task Scheduler

[**Noa**](https://noaai.app) is an AI task scheduler for macOS.

## About

This repository contains the landing page for Noa, hosted at [noaai.app](https://noaai.app) via GitHub Pages.

Noa is a macOS app that lets you schedule prompts and execute them automatically using the AI provider of your choice 
(Claude, Codex, Gemini, Ollama).
